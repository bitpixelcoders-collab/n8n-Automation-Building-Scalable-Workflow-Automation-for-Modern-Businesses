# What Is n8n?

n8n is a workflow automation platform that allows users to connect applications, APIs, databases, and cloud services using visual workflows.

Unlike many automation platforms, n8n provides:

* Visual workflow builder
* Self-hosting support
* Custom JavaScript and Python code
* API integrations
* AI-powered workflows
* Flexible deployment options

Its combination of no-code simplicity and developer flexibility makes it suitable for startups, agencies, and enterprises alike. ([GitHub][1])
